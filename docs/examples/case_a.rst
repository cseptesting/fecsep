Experiment Definition
---------------------

The following example shows the definition of a test experiment of a simple
forecast against a simple catalog. The input structure of the experiment is:

::

    case_a
        ├── catalog.json
        ├── config.yml
        ├── model.dat
        └── region.txt

Here, the catalog is simple and contains only one event
